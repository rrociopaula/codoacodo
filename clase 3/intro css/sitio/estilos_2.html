SELECTORES

a traves de los selectores indicaremos a qué elementos se le implementan los estilos

SELECTOR UNIVERSAL

se menciona con "*" e indica que es de aplicacion para todos los elementos HTML

Selector de etiqueta

vistos en el primer ejemplo, indicamos a que elemento se le implementará el estilo, ej a los "h2"

Selector de clase

primero agregamos el atributo "class" el alemento o a los elementos que queremos que implemente el estilo

Selector por identificador

al igual que las clases, primero debemos agregar el atrinuto "Id" pero SOLO A UN ELEMENTO

IMPORTANTE: EL ID DEBE SER UNICO
